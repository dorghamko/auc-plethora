object

void

void holds the name of the property of the object to be de-serialized by the XML decoder The value of the property attribute is the name of the object's property. The value of the property attribute must be one of the following: classpathJars (the JARs of the application e.g. App1.jar) command (the command to be executed to run the application e.g. java App1) displayResourceType (the type of the display resource e.g. ICON) displayResourceUrlName (the URL of the display resource e.g. resource:///app1.png) exec (main class for application) iconFilename (path to icon image file) menuGroup (the group to which the application shortcut belongs) name (application name)
